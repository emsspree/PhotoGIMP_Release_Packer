# PhotoGIMP 3 Release Packer

Create a ZIP-archived release package from the PhotoGIMP repository.

## photogimp_3.0_release_packer.bash

This script takes files from `📦/.config/GIMP/3.0` („old“ structure).

Usage:

1. Change to your local PhotoGIMP repository directory.
2. Run the script. The ZIP file will be created in the parent directory.
